### Why This Repo Exists
TurboUtil is built for developers who want fast, secure, offline alternatives to tools like jwt.io, quicktype.io, or jsonformatter.org ‚Äî without sending sensitive data to servers.

We use this repo to:
- Collect bug reports
- Gather ideas for new tools (e.g., YAML Formatter, Password Generator, GraphQL to TS...)
- Discuss improvements (UI, performance, new features)
- Share user stories/use cases

1. **Found a bug?** ‚Üí Open an issue with details (browser, steps to reproduce, screenshot if possible).

3. **Have feedback?** ‚Üí Post in Issues or Discussions tab (enable Discussions if you want threaded convos).
4. **Love the project?** ‚Üí Star ‚≠ê this repo, share on Twitter/Reddit, or mention "TurboUtil" when recommending offline dev tools.
